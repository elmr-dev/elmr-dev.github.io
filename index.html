elmr.dev

Ham radio software worth using — modern, cross-platform, built to last.

Ham software has solved a lot, but most of it stayed on one platform while everything else moved on. We're rethinking the tools people use every day — modern, cross-platform, built to last.

And we're doing it in the open, AI-assisted, by a small team that ships. The packages we build, how we build them, what we figure out along the way — model development, working code, tutorials — shared as we go, so the next person has a shorter path than we did.

An Elmer brings people along — that's the whole point.
